liblame

  • liblameDecoder

    Events:

    • DecodeComplete (Success As Boolean)

    Methods:

    • Decode
      Start the decoding process. The event DecodeComplete(Success As Boolean) will be raised on completion.
    • Initialize (EventName As String)
      Initialize the Decoder with an event name.
    • IsInitialized As Boolean
    • SetFiles (InputFilename As String, OutputFilename As String) As Boolean
      Set the source mp3 and destination wav file names. Returns True on success, False on error.

  • liblameEncoder

    Events:

    • EncodeComplete (Success As Boolean)

    Fields:

    • LAME_PRESET_DEFAULT As Int
    • LAME_PRESET_EXTREME As Int
    • LAME_PRESET_MEDIUM As Int
    • LAME_PRESET_STANDARD As Int

    Methods:

    • Encode
      Start the encoding process. The event EncodeComplete(Success As Boolean) will be raised on completion.
    • Initialize (EventName As String)
      Initialize the Encoder with an event name.
    • IsInitialized As Boolean
    • SetFiles (InputFilename As String, OutputFilename As String) As Boolean
      Set the source wav and destination mp3 file names. The Encoder will default to Quality of Encoder.LAME_PRESET_DEFAULT. Returns True on success, False on error.
    • SetQuality (Quality As Int)
      Set the encoding quality. Choose from: Encoder.LAME_PRESET_DEFAULT Encoder.LAME_PRESET_MEDIUM Encoder.LAME_PRESET_STANDARD Encoder.LAME_PRESET_EXTREME
